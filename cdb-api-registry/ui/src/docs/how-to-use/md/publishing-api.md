# Publishing your first API

This guide walks you through publishing an API to the Registry.

## Define metadata
Provide a clear name, version, owner, tags, and a short description.

## Upload OpenAPI spec
- Supported formats: YAML or JSON
- Validate your schema using the built-in validator

## Publish
Once validated, click Publish. Your API appears in the Marketplace.

## Tips
- Keep versions semantic (e.g., 1.0.0)
- Add meaningful tags for discovery
